2. Добавление аватара - должна появиться папка: media/users/user_id/avatar/filename
3. Удаление аватара - удаляется файл
4. Удаление пользователя - удаляется папка user_id (не работает при удалении через админку, а при удалении через shell - работает)

5. Создание объекта PhotoUser - должна появиться папка: media/users/user_id/photos/filename
6. Удаление объекта PhotoUser - должен удалиться файл filename (не работает при удалении через админку, а при удалении через shell - работает)

7. Создание объекта Post
8. Cоздание объекта PhotoPost - должна появиться папка: media/users/user_id/posts/post_id/filename
